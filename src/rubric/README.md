# Rubric subsystem

This package owns rubric generation and generated runtime configuration.

## Layout

- `templates/legacy.yaml`: rubric used by the earlier rubric experiment.
- `templates/generated_v2.yaml`: rubric generated by the current distillation
  pipeline.
- `generation/`: generation code, supervisor script, and prompts.
- `artifacts/instances/`: ignored per-question JSONL candidates.
- `artifacts/distilled/`: ignored distilled YAML and raw model responses.
- `artifacts/logs/`: ignored generation logs.
- `artifacts/archive/`: ignored historical backups.

Run generation modules from the repository root:

```bash
python -m src.rubric.generation.generate_instance --model qwen
python -m src.rubric.generation.distill --sample-per-qtype 40
```

Distillation and combination update both
`artifacts/distilled/rubric_templates_v2.yaml` and the runtime
`templates/generated_v2.yaml`. It never overwrites `templates/legacy.yaml`.

Select the template for an experiment with:

```bash
python experiments/tuning/veil_27b.py --rubric-template legacy
python experiments/tuning/veil_27b.py --rubric-template generated_v2
```
